Programma per la gestione delle spese domestiche.

Istruzioni per l'installazione:
crea un ambiente virtuale con il comando: python -m venv venv
attiva l'ambiente virtuale con il comando: .\venv\Scripts\activate
installa le dipendenze con il comando: pip install -r requirements.txt
avvia l'applicazione con il comando: python app.py
oppure: .\venv\Scripts\python.exe app.py 

Istruzioni per l'uso: 
- Aprire il browser e andare su http://localhost:5000

Basi di dati: 
- SQLite

API: 
- Google Gemini

Framework: 
- Flask

Interfaccia (responsive): 
- HTML
- CSS
- JavaScript
